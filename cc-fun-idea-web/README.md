# 交易員壓力測試：限時決策

一款純前端的交易知識限時問答遊戲，測試你在壓力下的交易決策能力。

## 玩法

- 共 10 題，開始前可選擇難度
- 題型涵蓋：漲跌幅計算、停損倉位、風險報酬比、滑價判斷
- 答對每題得 10 分，答錯或超時不得分
- 每題四個選項會隨機排列，避免固定位置偏差
- 結束後依分數評定等級：新手 / 穩健 / 老司機

## 難度分級

| 等級 | 每題限時 | 說明 |
|------|----------|------|
| 🟢 入門 | 20 秒 | 充裕的思考時間，適合初學者 |
| 🟡 專家 | 10 秒 | 中等壓力，需要快速反應 |
| 🔴 快手 | 5 秒 | 極限挑戰，模擬真實盤中壓力 |

遊戲畫面會顯示目前選擇的難度等級與倒數秒數，結果頁也會標示本次使用的難度。

## 使用方式

直接用瀏覽器開啟 `index.html` 即可遊玩，無需伺服器。

```bash
# macOS
open index.html

# Linux
xdg-open index.html

# 或直接拖進瀏覽器
```

## 技術

- 純 HTML / CSS / JS，單一檔案
- 無外部依賴、無 CDN
- 支援手機直向操作
- 每題選項會隨機排列，正確答案位置不固定
