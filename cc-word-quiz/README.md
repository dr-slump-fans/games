# 📝 英文單字測驗

一個純前端的英文單字測驗遊戲，測試你的英文詞彙量！

## 玩法

1. 選擇題庫（常用 3000 單 / 常用 7000 單）
2. 選擇難度等級
3. 點擊「開始測驗」
4. 畫面會顯示一個英文單字，從 4 個中文選項中選出正確意思
5. 每題答完會立即顯示對錯與簡短解釋
6. 共 10 題，最後顯示總分與等級

## 題庫說明

| 題庫 | 檔案 | 說明 |
|------|------|------|
| 📗 常用 3000 單 | `data/words_3000.json` | 基礎常用單字 |
| 📘 常用 7000 單 | `data/words_7000.json` | 進階 / 學測指考等級 |

題庫以外部 JSON 檔案載入，方便擴充與替換。

## 難度說明

| 難度 | 作答時間 | 適合對象 |
|------|----------|----------|
| 🟢 入門 | 20 秒 | 初學者，可以慢慢想 |
| 🟡 專家 | 10 秒 | 有一定基礎，考驗反應 |
| 🔴 快手 | 5 秒 | 高手挑戰，瞬間判斷 |

## 計分方式

- 答對：+10 分
- 答錯 / 超時：0 分
- 滿分：100 分

## 等級評定

| 分數 | 等級 |
|------|------|
| 90 以上 | 🏆 單字達人 |
| 70–89 | 🌟 進階學習者 |
| 50–69 | 💪 穩步前進 |
| 50 以下 | 🌱 初學者 |

## 開啟方式

需透過 HTTP 伺服器開啟（因為使用 `fetch` 載入 JSON 題庫）：

```bash
# 使用 Python 內建伺服器（推薦）
cd cc-word-quiz
python3 -m http.server 8080
# 然後開啟 http://localhost:8080

# 或使用 Node.js npx
npx serve .
```

## 如何替換成真實 3000 / 7000 題庫

目前 `data/words_3000.json` 和 `data/words_7000.json` 各含約 60 題示例資料。若要替換成完整題庫，只需按照以下格式準備 JSON 檔案：

### JSON 格式

```json
[
  {
    "word": "abandon",
    "pos": "動詞",
    "phonetic": "/əˈbændən/",
    "answer": "放棄；遺棄",
    "opts": ["放棄；遺棄", "豐富的", "吸收", "完成"],
    "tip": "abandon ship 是「棄船」的意思。"
  }
]
```

### 欄位說明

| 欄位 | 類型 | 說明 |
|------|------|------|
| `word` | string | 英文單字（題目顯示） |
| `pos` | string | 詞性（名詞、動詞、形容詞等） |
| `phonetic` | string | 音標（KK / IPA） |
| `answer` | string | 正確中文翻譯 |
| `opts` | string[] | 4 個中文選項（必須包含 answer） |
| `tip` | string | 學習提示 / 例句 |

### 替換步驟

1. 準備好符合上述格式的 JSON 檔案
2. 將檔案放到 `data/` 目錄下，檔名為 `words_3000.json` 或 `words_7000.json`
3. 確認 JSON 格式正確（可用 `python3 -m json.tool data/words_3000.json` 驗證）
4. 重新開啟頁面即可

## 技術說明

- 純前端：HTML + CSS + JS，單一 HTML 入口
- 題庫：外部 JSON 檔案，透過 fetch 載入
- 無外部依賴（無 CDN）
- 手機友善，響應式設計
- 每次隨機抽 10 題，選項與題目皆隨機排列
